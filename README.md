## Описание программы

** Movie Library ** - это GUI-приложение для хранения информации о фильмах с фильтрацией. Добавленные фильмы добавляются в JSON.

### Возможности:
- ✅ Добавление фильмов
- ✅ Фильтрация добавленных фильмов
- ✅ Удаление выбранных фильмов

## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter 
-  Сохранение в JSON

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
3. Открыть файл Movie Library.py
4.Запустить приложение
   















## Примеры использования

###Добавление фильма
1.	Заполните поля:

-	Название ("Универ")

-	Жанр ("Комедия")

-	Год (2025)

-	Рейтинг (9)

Нажмите "Добавить фильм"

Фильм появится в таблице

##Фильтрация
-По жанру: Введите часть жанра (например "ком" - найдет "Комедия")

-По году: Введите год 


##Удаление
-Выделите фильм в таблице

-Нажмите "Удалить выбранный фильм"

-Подтвердите удаление
